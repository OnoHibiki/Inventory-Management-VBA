README

◇Excelとaccessを用いた在庫管理アプリです。
（office365使用　accsesは）

〇「メイン」シート
・商品名（部分一致）、商品ID（完全一致）での商品検索ができます。
 「,」で区切れば、複数条件のしても可能です。

・入出庫機能
　商品IDを指定して数量を入力。入庫or出庫を押すと、在庫数を変更できます

〇「在庫情報一覧」シート
・在庫データベースにある情報を一覧で取得できます。
・DBへの商品情報の追加、削除ができます。削除の場合のみ、確認ポップアップがでます。
　追加、削除後即反映です。削除した場合、商品IDは一意なので飛び番号が生まれます。

〇「入出庫履歴」シート
・入出庫の履歴が一覧表示されます。
・「メイン」シートで入出庫が行われた場合、連動して自動で履歴が追加されます

〇DBカラム情
・ACCESSのカラム情報です
